
# ToDo-List

ToDo list is a simple web application to save Your daily tasks in order not to miss anything.

### Information

realization of a toDoList basic to train these skills

### Features

- Add new task
- Mark task as completed / uncompleted
- Edit task
- Delete single task

### Technologies used:

- JavaEE
- JSP
- Spring Boot
- Jpa hibernate / MySql
- Bootstrap
